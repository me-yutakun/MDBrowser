## [1.0.0] — 2026-04-26
### Added
- Split-pane layout with responsive sidebar and Markdown viewer
- Local-first, 100% offline-capable architecture
- Folder-level loading and single-file drag-and-drop support
- Live preview instant auto-refresh on local file save
- Support for Mermaid diagrams and KaTeX math
- Support for GitHub Alerts (Note, Tip, Important, etc.)
- Infinite-depth collapsible file tree with automatic `.md` filtering
- Intelligent smooth-scroll navigation for all heading anchors
- Persistent Light/Dark themes (preserves scroll position)
- Floating "Back to Top" navigation button
- Fully documented codebase with JSDoc and architecture diagrams
