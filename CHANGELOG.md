# Changelog
All notable changes to this project will be documented here.

---

## [1.0.0] — 2026-04-24
### Added
- Split‑pane layout (left file browser, right Markdown renderer)
- Infinite-depth collapsible folder tree with MD‑only filtering
- Local-first architecture (fully offline capable via `ext/` folder)
- GitHub‑style Markdown rendering with light/dark themes
- Single-button collapsible sidebar toggle
- Mermaid diagram rendering (`mermaid` code blocks)
- KaTeX math rendering (`$...$`, `$$...$$`)
- Drag & drop support for single `.md` files
- Chrome‑yellow link styling for better visibility
- Demo scripts and documentation
- Developer onboarding, architecture diagrams, and disclaimers in README
- Fully commented HTML application core
